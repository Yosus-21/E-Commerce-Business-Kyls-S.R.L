# Routes Directory

Este directorio contendrá las definiciones de rutas de la API:

- `authRoutes.js` - Rutas de autenticación
- `productRoutes.js` - Rutas de productos
- `categoryRoutes.js` - Rutas de categorías
- `cartRoutes.js` - Rutas del carrito
- `orderRoutes.js` - Rutas de pedidos
- `userRoutes.js` - Rutas de usuario

Las rutas se crearán en el siguiente paso del desarrollo.
